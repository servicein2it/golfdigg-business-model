# Golfdigg — Business Model Presentation

An interactive, slide-deck **business model & competitive analysis** of **Golfdigg**, Thailand's No.1 golf tee-time booking platform — built to understand the incumbent and map the opportunity to build a better competitor.

**▶ Live deck:** https://servicein2it.github.io/golfdigg-business-model/

## About

A single, self-contained HTML file (`index.html`) — no build step, no dependencies. 25 slides in two acts:

- **Part 1 · How Golfdigg works** — a plain-language explainer ("it's Booking.com for golf") then the full model: Problem → Solution → Product → Business model → Revenue → Unit economics → Market → Customers → Traction → Flywheel → Moat → Funding → Team.
- **Part 2 · Analysis & opportunity** — SWOT at a glance → What they have (strengths) → What they need (gaps) → Opportunities → **Where a better product wins** (a "Golfdigg today vs. a better product" comparison) → Vision.
- **Navigation:** arrow keys `← →` (or `Space` / `PageUp`/`PageDown`), `Home`/`End`, on-screen prev/next + slide counter, progress bar, swipe on touch devices.
- **`O`** — slide overview grid · **`F`** — fullscreen · **`T`** — light/dark theme.
- Fully responsive, light & dark themes, reduced-motion friendly, and degrades to a plain vertical scroll if JavaScript is unavailable.
